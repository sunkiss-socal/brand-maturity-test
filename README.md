## 这是什么？

这是一份面向 **中国 B2B 企业决策者** 的品牌出海美国认知评估工具。8道题，每道题对应一个真实的出海陷阱，帮你快速定位自己当前处于哪个认知阶段。

## 测试涵盖哪些维度？

| 维度 | 考察重点 |
|------|---------|
| 品牌定位认知 | 你认为的核心竞争力，真的是美国市场买单的吗？ |
| 买家决策逻辑 | 美国B2B采购的真实流程 vs 你以为的流程 |
| 数字资产认知 | 你的品牌在美国市场的"数字存在"够不够？ |
| 信任构建认知 | 中国品牌在美国建立信任的正确方式 |
| 本地化深度认知 | "翻译成英文"和"真正的本地化"差多远？ |
| 合规与风险认知 | 合规是后台工作还是战略基础设施？ |
| 营销投入认知 | 没有信任基础的流量投入 = 往漏水的桶里倒水 |
| 战略节奏认知 | 出海美国的时间预期：6个月 vs 3-5年 |

## 评分体系

- 满分 **100分**，4个认知阶段：
  - 🔴 **认知探索期**（25-40分）— 还在用中国地图找美国的路
  - 🟡 **认知觉醒期**（41-60分）— 知道规则不同，但还没找到航线
  - 🟢 **认知成熟期**（61-80分）— 看清了航线，需要领航员
  - 🔵 **认知领先期**（81-100分）— 少数清醒者，让认知优势变成执行优势
